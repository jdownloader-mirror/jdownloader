Curl Router Reconnect version 0.2.7

Mit diesem Programm kannst Du schnell und einfach Deinen Router reconnecten. 
Zur Zeit werden über 84 Router unterstützt. Diese Liste habe ich teilweise von Usern bekommen und den Rest im Internet gefunden.
Wenn irgendein Fehler integriert ist einfach schreiben. Dieses Programm lebt von eurer Unterstützung.

Ihr habt einen eigenen neuen Router oder ein besseres Script?

Im CAB Archiv progs.cab befinden sich Programme die einige Router und Routinen benötigen.

Die Datenbank wird öfters erneuert als dieses Programm. Deshalb gucke einfach von Zeit zu Zeit auf meine Seite ob eine neue Datennbank Version verfügbar ist.

Wähle Deinen Router aus und passe folgende Daten an
IP, einige Router benötigen ein Password.

Add sleep. Pause integrieren z.b 1000 = 1 sekunde warten
Add printip routine. Es wird eine Routine integriert die vor dem Reconnect die IP abfragt und nach dem reconnect und wenn die identisch ist wird der Router erneut reconnect. Printip ist ebenfalls ein kleines Tool was ich geschrieben habe. Es fragt nur checkip.dyndns.org ab.

Wenn Du mir ein neues Script schickst dann sollte es bei Dir einwandfrei funktionieren.


Wie könnt Ihr selber ein Script erstellen.
Naja das einfachste ist es, Du öffnest die entsprechende HTML Seite im Routermenü wo Du Dich trennen und verbinden kannst.
Nun einfach den Source angucken was passiert wenn der entsprechende Button gedrückt wird.
